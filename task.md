# Техническое задание: Telegram Mini App «Flip a coin»

## 1. Назначение

Mini App в Telegram: интерактивная **золотая монета** (визуал на **CSS и градиентах**), стороны **Heads** и **Tails**. Подброс по кнопке **Flip a coin** и/или по тапу на монету: вращение **3 секунды**, затем остановка на случайном исходе.

## 2. Локализация

- Интерфейс **только на английском** (кнопки, подписи, Heads/Tails).

## 3. Интеграция с Telegram

- Использовать **Telegram Mini Apps API** (официальный скрипт или `@twa-dev/sdk`).
- **Обязательно:** учёт **темы Telegram** — цвета из `themeParams` (фон, текст, кнопки и т.д. по смыслу экрана), чтобы приложение выглядело согласованно со светлой/тёмной темой клиента.
- Вызов `WebApp.ready()` после готовности UI.

## 4. Функциональные требования

| ID  | Требование                                                                                      |
| --- | ----------------------------------------------------------------------------------------------- |
| F1  | Золотая монета средствами **CSS/градиентов** (без обязательных внешних ассетов).                |
| F2  | Стороны: **Heads** и **Tails**.                                                                 |
| F3  | Запуск: кнопка **Flip a coin** и тап по монете; во время анимации повторный запуск блокируется. |
| F4  | Анимация вращения ~**3 с** (целевое значение 3000 мс).                                          |
| F5  | Случайный исход **Heads** или **Tails** (равная вероятность, если не оговорено иное).           |

## 5. Стек

| Компонент | Выбор                                                                                            |
| --------- | ------------------------------------------------------------------------------------------------ |
| Язык      | **TypeScript**                                                                                   |
| UI        | **React**                                                                                        |
| Сборка    | **Vite**                                                                                         |
| Стили     | **Tailwind CSS** (+ при необходимости `@tailwindcss/vite` / PostCSS по шаблону Vite)             |
| Telegram  | `telegram-web-app` / **@twa-dev/sdk**                                                            |
| Деплой    | **Vercel** (пользовательский домен не требуется; достаточно URL вида `*.vercel.app` с **HTTPS**) |

## 6. Деплой (Vercel)

- Репозиторий подключается к Vercel; для Vite — **build command** `npm run build` (или `pnpm`/`yarn` по проекту), **output** — каталог `dist` (стандарт для Vite).
- В настройках бота (BotFather) указывается **публичный HTTPS-URL** деплоя Mini App.
- Домен покупать не требуется.

## 7. Критерии приёмки

1. Открытие в Telegram, `ready()`, визуал согласован с темой Telegram.
2. Английский UI; монета и логика подброса по ТЗ.
3. Успешная сборка и работа с деплоя на Vercel.

## 8. Исходная формулировка (task.md)

Создать Telegram Mini App: Flip a coin. В интерфейсе телеграма должна отображаться золотая монета с надписью Heads на одной стороне и tails на другой стороне. При нажатии на кнопку Flip a coin или на саму монетку должна запускаться анимация крутящейся монеты. Монета должна крутиться 3 секунды и останавливаться рандомно на heads или tails.
